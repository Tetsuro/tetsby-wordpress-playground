## 1. Install plugin for menu

## 2. Create an .env file

Create a `.env.development` file.

Add the following:

```
GATSBY_WP_BASE_URL=localhost:XXXX
GATSBY_WP_PROTOCOL=http
```

Add these to through the Settings in Netlify too!

Fresh Start

Man, it’s been a while since I last posted! Time flies!

July was really shitty to be honest. I done broke up with the special lady, so that kept my mind occupied for a while. You know how it is, when you break up you pretty much lose motivation to do anything haha… though I’ve been feeling much better lately. I usually don’t write about this kinda stuff here… but it’s my blog, so whatevuh I do what I want!

So yeah, I’ve started up some new projects to keep me busy. One of the things that really got my juices flowin’ was the launch of the Merchify app. Merchify is this new Shopify app that lets you print and sell t-shirts, mugs, and posters on-demand, so you don’t have to worry about any leftovers. The plan is to start up my own Shopify store and sell some t-shirt designs with my doodles 🙂

Check out Merchify! It’s fucking sick!

Another thing I want to work on is tutorials on Shopify’s new line item properties feature. This feature is seriously huge! It opens up a lot of opportunities in Shopify, and makes it that much more powerful. You can read more about line item properties in the Wiki here. You can see a demo here. I’m thinking tutorials on how to implement a colour picker to a product page, or even color swatches… the possibilities are endless!

Finally, starting to get back into playing live music again. The band that I started last year, Move Your Mountain, is unfortunately on hiatus until next April, but I’m going to be hitting up open jams around the city. Local jazz singer Renée Yoxon runs jazz jam sessions at Pressed, so I’m going to be attending that on the 18th. I’m also going to be hitting up the open blues jams at The Rainbow, pretty stoked for that too!

Come check it out!

Gonna wrap this post up with a song that I discovered through my friend and coworker Dan Eveleigh, who runs this wicked blog called My Life in Tunes. The tune is called “Getaway” by Dr. John, and it’s basically my theme song right now, hahaha 😀
